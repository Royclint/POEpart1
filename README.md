## Description
This project implements the VOX808 messaging system, combining Part 1 registration/login validation with Part 2 message handling and JUnit testing.

## Features
- User registration (Name, Surname, Username, Password, Phone Number)
- Login validation (underscore + password complexity)
- Messaging system (send multiple messages, view sent messages)
- File handling with JSON storage
- JUnit tests for Login and Message classes

## How to Run
1. Open in NetBeans.  
2. Clean and Build (Shift + F11).  
3. Run (F6).  
4. Follow on‑screen prompts.

## How to Test
1. Right‑click `LoginIT.java` → Test File.  
2. Right‑click `MessageIT.java` → Test File.  
3. Ensure all tests pass (green bars).
